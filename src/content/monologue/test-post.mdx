Встроенный код: `console.log("Hello World!")`

Блок кода с подсветкой синтаксиса:

```jsx
function Counter() {
  const [count, setCount] = useState(0);
  return (
    <div>
      <button onClick={() => setCount((c) => c + 1)}>
        Кликнуто: {count} раз
      </button>
    </div>
  );
}
```

Python пример:

```python
def fibonacci(n):
    a, b = 0, 1
    for _ in range(n):
        yield a
        a, b = b, a + b
```
